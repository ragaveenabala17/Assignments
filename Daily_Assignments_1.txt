Assignment: Armstrong Number Checker (Java)

Write a Java program to check whether a given number is an Armstrong number or not.

ðŸ‘‰ A number is called an Armstrong number if the sum of the cubes of its digits is equal to the number itself.
Example:

=153 âœ… (Armstrong number)
123 â†’ Not an Armstrong number âŒ


solution : 
-------------
package assignment;

public class Armstrong {

	public static void main(String[] args) {
		//TODO Auto-generated method stub
		
		 int number = 153; // You can change this value
		        int originalNumber = number;
		        int sum = 0;

		        while (number !=0) {
		            int digit = number % 10;
		            sum += digit * digit * digit; // cube of digit
		            number = number / 10;
		        }
		        if (sum == originalNumber) {
		            System.out.println(originalNumber + " is an Armstrong number.");
		        } else {
		            System.out.println(originalNumber + " is NOT an Armstrong number.");
		        }
		    }
	}

----------
